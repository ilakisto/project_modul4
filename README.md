Для создания некоторых из анимаций использовался искусственный интеллект (а именно claude ai), в частности:
- на странице "where to go" ии использовалась для появления текста при увеличении картинки
- в попапе с бронью тура ии использовалась для распределения домов в зависимости от выбранного климата
- для создания подчеркивания в менюбаре при переходе на разные страницы
